About me

I'm Bharath, a backend developer working on Java and Spring Boot systems in the banking domain. Most of my day is spent inside services that talk to each other over Kafka rather than waiting on each other over REST — and most of what I write here comes directly out of that work.

This log exists because I learn best by writing things down properly once, instead of re-Googling the same Kafka consumer property for the fifth time. If a reference guide saves me twenty minutes at 2 AM, it's worth publishing.

Outside of backend work, I'm building NeoBank — a personal event-driven microservices project — as a way to go deep on the same ideas I write about here: Kafka, Cassandra, and now a bit of Spring AI.
