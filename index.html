About me

I write about backend systems the way I'd want them to be explained precisely, and without the fluff.

I'm Bharath, a backend developer working on Java and Spring Boot systems in the banking domain. Most of my day is spent inside services that talk to each other over Kafka rather than waiting on each other over REST, and most of what ends up here comes straight out of that work.

This log exists because I learn best by writing things down properly once, instead of re-searching the same thing again and again. If a reference guide saves someone twenty minutes during an incident, it's worth publishing.

Outside day-to-day work, I build NeoBank - a personal event-driven microservices project - to go deep on the same ideas I write about here: Kafka, Cassandra, and lately, Spring AI.
